# Instructions

## Setup

1. Install **Node.js v20+** or **bun v1.0.20**.
2. Switch to the `step/1` branch.
3. Install dependencies:
   ```bash
   npm install # or bun install
   ```
4. Create a `.env` file and add your OpenAI API Key:
   ```
   OPENAI_API_KEY='YOUR_API_KEY'
   ```

## Run the Project

```bash
npm start
```

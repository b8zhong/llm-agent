# Instructions

## Setup

1. Install dependencies:
   ```bash
   npm install # or bun install
   ```
2. Create a `.env` file and add your OpenAI API Key:
   ```
   OPENAI_API_KEY='YOUR_API_KEY'
   ```

## Run the Project

```bash
npm start
```
